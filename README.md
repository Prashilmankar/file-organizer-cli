# ?? Smart File Organizer CLI
A lightning-fast, zero-dependency CLI utility built with Node.js to automatically organize messy directories.
## ?? Features
- **Instant Organization:** Sorts loose files into dedicated folders (`Images`, `Documents`, `Archives`, `Code`).
- **Smart Scoping:** Defaults to current directory or accepts a custom target path.
- **Built-in Safety:** Never moves its own configuration or core source files.
## ??? Usage
Run `npm link` inside the directory, then use `organize` or `organize /path/to/folder` anywhere.
